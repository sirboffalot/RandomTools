Windows DNS Server Version Converter

How to use

This tool converts the hex version number of a Microsoft DNS server to something useful. IF searched for in nmap, it will look like this:

			sudo nmap -sSU -p 53 --script dns-nsid 1.1.1.1
			|_  bind.version: Microsoft DNS 6.1.7601 (1DB15F75)

This tool converts that last bit to a readable version number (like Nessus does).
